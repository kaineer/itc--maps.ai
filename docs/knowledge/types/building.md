# Здание

## Здание, определенное точками, полученными из openmaps

* **Название**: MapsBuilding
* **Поля**: 
  * outline: Point[] -- список точек, образующих контур здания. Значимыми координатами являются, скорее всего, только x и z
  * height: number -- высота в метрах (предположительная)
  * address: string -- адрес в городе

## Здание, загружаемое как модель из блендер

* **Название**: ModelBuilding
* **Поля**:
  * model: minio url -- или любое другое средство, позволяющее подгрузить модель
  * position: Point -- может первоначально инициализироваться позицией здания из карт
  * rotation: Rotation -- в идеальном мире должен иметь значение (0, 0, 0)
  * scale: Scale -- опять же, в идеальном мире должен иметь значния (1, 1, 1)

* **Примечание**: Предполагается, что все модели будут экспортированы в формат fbx
* **Где почитать про FBX**: https://r3f.docs.pmnd.rs/tutorials/loading-models
