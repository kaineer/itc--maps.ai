# Экскурсии

## Точка экскурсии
* **Название**: TrackPoint
* **Поля**: 
 * markup: string -- разметка страницы с краткой информацией
 * position: Point
 * type: тип точки, в том числе, ограничения на тип точки (что конкретно тут будет, обсудим позже)

## Экскурсия
* **Название**: Track
* **Поля**:
 * name: string -- название экскурсии
 * points: TrackPoint[]
