# Основные типы объектов

## Координаты точки в пространстве

* **Название**: Point
* **Поля**:
  * x, y, z - number -- координата, 1 соответствует одному метру

## Масштабирование модели

* **Название**: Scale
* **Поля**:
  * x, y, z - number -- масштаб по соответствующей координате. Значение по умолчанию 1.0

## Поворот модели

* **Название**: Rotation
* **Поля**:
 * a, b, c - number -- поворот вокруг соответствующей (x, y, z) оси
