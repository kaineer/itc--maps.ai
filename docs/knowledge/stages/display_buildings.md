# Отображение зданий с использованием threejs

## Шаги по реализации

 * Создаем react + typescript приложение используя vite
 * Добавляем зависимость @react-three/fiber
 * Отображаем на весь экран компонент, внутри которого загружаем список зданий, которые выдает backend (см. предыдущую стадию, скорее всего, нужно будет сделать fetch('http://localhost:3000/buildings', {
   method: 'PUT',
   body: {
     position: { x, z } // x, z -- координаты здания на чкалова 3
   }
 })
 * После того, как загрузится список зданий, каждое здание отрисовываем как набор прямоугольников с высотой, полученной из атрибута height и основанием из двух точек, полученных из текущей и следующей точек полигона
 * ... будет дополняться
