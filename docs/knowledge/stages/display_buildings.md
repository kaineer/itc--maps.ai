# Отображение зданий с использованием threejs

## Шаги по реализации

 * Создаем react + typescript приложение используя vite
 * Добавляем зависимость @react-three/fiber
 * Отображаем на весь экран компонент, внутри которого загружаем список зданий, которые выдает backend (см. предыдущую стадию, скорее всего, нужно будет сделать 
```javascript
 fetch('http://localhost:3000/buildings', {
   method: 'PUT',
   body: {
     position: { x, z } // x, z -- координаты здания на чкалова 3
   }
 })
 ```
 * После того, как загрузится список зданий, каждое здание отрисовываем как набор прямоугольников с высотой, полученной из атрибута height и основанием из двух точек, полученных из текущей и следующей точек полигона
 * Перемещение камеры в приложении должно быть ограничено:
   * Высота камеры фиксирована на высоте 1.8
   * Перемещение должно быть реализовано назад и вперед (по клавишам w и s)
   * Поворот камеры вправо, влево, вверх и вниз, должны быть доступны через OrbitControl, при зажатой левой кнопке мыши
   * При этом в момент, когда пользователь собирается начать передвигаться вперед/назад, положение камеры должно снова становиться идеально горизонтальным (чтобы не было изменения высоты)
