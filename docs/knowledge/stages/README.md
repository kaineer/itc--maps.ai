# Шаги по отображению данных, полученных из открытых источников

 * import_maps.md -- описывает получение данных с внешнего сайта и генерацию json-файлов
 * check_buildings.md -- описывает создание html-файла, в котором можно посмотреть насколько валидны координаты зданий
 * serve_buildings.md -- описывает создание fastify-сервера, который отдает здания для отображения в браузере
 * display_buildings.md -- описывает создание фронтенд-части
